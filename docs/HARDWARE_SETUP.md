# ⚠️ ВАЖНОЕ ПРЕДУПРЕЖДЕНИЕ

**Все подключения к силовой части станка (220В/380В) должны проводиться квалифицированным персоналом при отключенном питании. Разработчик не несет ответственности за ущерб.**

## Подключение датчиков

### PZEM-004T (напряжение/ток)
- RX/TX модуля подключаются к GPIO16/GPIO17 (настройка в `config.json`).
- Используйте гальванически развязанный трансформатор тока и напряжения.
- Рекомендуется оптроны на линии UART (например, 6N137) и разделение земли.

### MPU6050 (вибрация)
- I2C: SDA=GPIO21, SCL=GPIO22 (по умолчанию).
- Подтяжки 4.7 кОм к 3.3В.

### DS18B20 (температура)
- DATA=GPIO4, питание 3.3В, подтяжка 4.7 кОм между DATA и 3.3В.

### YF-S201 (датчик потока)
- Сигнал на GPIO32 (INPUT_PULLUP).
- Сигнал можно дополнительно развязать оптопарой PC817.

### ACS712 (ток двигателей)
- Аналоговый выход на GPIO34.
- Питание модуля 5В, выход через делитель до 3.3В.

### Реле аварийной остановки
- GPIO25 -> оптроны -> силовое реле.
- Обязательна гальваническая развязка.

## Номиналы и развязка
- Делитель для 5В -> 3.3В: 10к + 20к.
- Оптроны: PC817 (медленные), 6N137 (быстрые UART).
